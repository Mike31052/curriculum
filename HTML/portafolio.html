PROYECTOS REALIZADOS

  • FUD:

    Formato Unico de Declaración, es un sistema web que le permite a las instituciones gubernamentales realizar el registro de las víctimas que tienen o han tenido algún derecho violentado, además de darle seguimiento al proceso de restauración de derechos con la institución correspondiente.


  • Sistema para validar la calidad del agua:

    Consta de 5 sensores que miden los parametros del agua (PH, cloro, flujo, etc.) y los valores obtenidos son procesados por un microcontrolador, el cual los procesa y los envia a un modulo bluetooh, el cual a su vez los transmite a un dispositivo movil.


  • Sistema web para el control y gestión de tuberias de distribicuón de agu en la CDMX:

    Sistema web que le permite a los consumidores de este servicio reportar fugas de agua, además de realizar sus pagos de agua. A los administradores les permite ver las ubicasiones de las fugas que existen y conocer la red de distribución, además de estimar cuales son las fugas más grandes para así poder actuar.


  • Aplicación movil para recuperación del covid:

    Te muestra ejercicios respiratorios que puedes hacer para una mejor recuperación del covid-19.
